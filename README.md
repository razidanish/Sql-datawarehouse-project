# Sql-datawarehouse-project

**Project Overview**
This project demonstrates the design of a SQL Data Warehouse using **Medallion Architecture** with Bronze, Silver, and Gold layers. The focus is on organizing raw datasets into structured tables for efficient querying and analysis.

**Features**
- Designed a **three-layer data warehouse**:  
  - **Bronze Layer:** Raw data storage  
  - **Silver Layer:** Cleaned and organized tables  
  - **Gold Layer:** Aggregated tables for analysis
- Created **SQL Views on Gold layer** to simplify queries and support analysis.
- Performed **SQL-based data analysis** to extract insights and identify trends.
- Supported efficient reporting and querying for business decisions.

**Technologies Used**
- SQL Server Management Studio (SSMS)
- SQL

**Project Structure**
**Bronze/*      Raw data tables
**Silver/*      Cleaned tables
**Gold/*        Aggregated tables for analysis
**Dataset/*     Original datasets
**Scripts/*     SQL queries for analysis
